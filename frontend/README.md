# Dashboard AutoFollowBling

Dashboard em React + TypeScript para exibir estatísticas de mensagens WhatsApp enviadas pelo backend AutoFollowBling.

## Funcionalidades

- Resumo de mensagens enviadas por dia, semana e mês
- Gráficos de sucesso e erro
- Tabela de envios recentes

## Como usar

1. Configure o backend para expor os endpoints de estatísticas.
2. Execute o frontend com:
   ```
   npm run dev
   ```
3. Acesse o dashboard em http://localhost:5173

## Customização

- O frontend consome endpoints REST do backend. Ajuste a URL base em um arquivo `.env` ou diretamente no código se necessário.
